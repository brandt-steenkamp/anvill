# Anvill

Anvill is a macOS-first Braille translation project designed to make Braille tooling more open, accessible, and extensible.

## Current status

The core prototype is working.

Anvill can currently:

- accept pasted text input
- process text through Liblouis
- return ASCII Braille output

## Technology

- Swift
- SwiftUI
- Liblouis integrated directly into the project using a bridging header

## Goals

- build a practical Braille translation workflow for macOS
- support file import and export
- improve accessibility tooling for blind users and transcribers
- provide an open-source platform for Braille development
